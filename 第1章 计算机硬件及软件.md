## ✍️ 长句分析

### Sentence 1
> The physical components of a computer system, including the motherboard, processor, and various peripheral devices, are collectively referred to as hardware.

**计算机系统的物理组件，包括主板、处理器以及各种外围设备，统称为硬件。**

- **主句：** The physical components are collectively referred to as hardware.（物理组件被统称为硬件）
- **插入语：** including the motherboard, processor...（包括主板、处理器…），作补充说明
- **介词短语：** of a computer system，修饰 physical components
- **重点词汇：** physical components（物理组件）、motherboard（主板）、processor（处理器）、peripheral devices（外围设备）、collectively（共同地/总体地）

> 💡 外围设备（Peripherals）在传统认知中通常指连接到 PC 的键盘、鼠标或打印机等实现人机交互的工具。但在广义的专业英语与现代智能计算语境下，任何独立于核心计算单元之外、负责数据输入或输出的设备都统称为"外围设备"。

---

### Sentence 2
> The central processing unit (CPU), often considered the brain of the computer, is responsible for fetching, decoding, and executing instructions sequentially.

**中央处理器（CPU）通常被认为是计算机的"大脑"，负责按顺序获取、解码并执行指令。**

- **主句：** The CPU is responsible for fetching, decoding, and executing instructions.
- **同位语：** often considered the brain of the computer（常被认为是计算机的"大脑"）
- **动名词并列：** fetching, decoding, and executing（对应 CPU 工作中的取指、译码和执行三个核心环节）
- **重点词汇：** central processing unit（中央处理器）、fetch（获取）、decode（解码）、execute（执行）、sequentially（按顺序地/串行地）

> 💡 这里要特别注意 sequentially。CPU 最突出的特点之一，就是擅长按照步骤连续处理复杂指令，也就是进行串行逻辑控制。这也是为什么 CPU 适合承担系统控制、程序调度和复杂判断任务。相对地，如果面对深度学习训练或大规模数值计算这类高度并行的任务，就往往要借助 GPU 这样的并行架构。

---

### Sentence 3
> Primary storage, typically referring to RAM, is volatile and holds data temporarily, whereas secondary storage provides permanent data retention.

**主存储器（通常指 RAM）是易失性的，只能临时保存数据；而辅助存储器则提供永久的数据保留。**

- **主句：** Primary storage is volatile and holds data temporarily.
- **对比连词：** whereas（强调主存和辅存之间的对比）
- **插入定语：** typically referring to RAM（这里的 primary storage 通常就是指 RAM）
- **重点词汇：** primary storage（主存储器）、secondary storage（辅存储器）、volatile（易失性的）、temporarily（临时地）、permanent（永久的）、retention（保留）

> 💡 Volatile 的意思是"断电后内容会丢失"，这正是 RAM 的典型特征。如果运行大型数据集或复杂程序时出现 Out of Memory，通常是 Primary storage / RAM 被占满了。

---

### Sentence 4
> System software, particularly the operating system, acts as a crucial interface between the user, the application software, and the underlying hardware.

**系统软件（尤其是操作系统）在用户、应用软件以及底层硬件之间充当着至关重要的接口。**

- **主句：** System software acts as a crucial interface.
- **同位语：** particularly the operating system（尤其强调操作系统）
- **介词短语：** between the user, the application software, and the underlying hardware
- **重点词汇：** system software（系统软件）、operating system（操作系统）、crucial（至关重要的）、interface（接口/界面）、underlying（底层的）

> 💡 这里的 interface 很关键。在软件工程语境里常翻译成"接口"；在用户操作层面也可以理解为"界面"。通过 underlying hardware 这个表达，可以看到计算机系统的层级关系：Hardware → System Software → Application Software → User。

---

### Sentence 5
> Application software consists of specialized programs designed to perform specific tasks for users, such as word processing, database management, or data analysis.

**应用软件由专门的程序组成，旨在为用户执行特定任务，例如文字处理、数据库管理或数据分析。**

- **主句：** Application software consists of specialized programs.
- **过去分词作定语：** designed to perform specific tasks for users
- **举例说明：** such as word processing, database management, or data analysis
- **重点词汇：** application software（应用软件）、consist of（由…组成）、specialized（专门的）、perform（执行）、specific tasks（特定任务）、word processing（文字处理）

> 💡 应用软件的核心价值不在于管理整个计算机系统，而在于帮助用户完成具体工作。比如写论文用的 Word、做统计用的 SPSS 或 Python，都属于应用软件。

---

### Sentence 6
> At the lowest level of hardware, all data and instructions are represented in binary form, utilizing combinations of zeros and ones known as bits.

**在硬件的最底层，所有数据和指令都以二进制形式表示，利用由 0 和 1 组成的组合，这些组合被称为"位"（bits）。**

- **介词短语作状语：** At the lowest level of hardware
- **主句：** all data and instructions are represented in binary form
- **现在分词作伴随状语：** utilizing combinations of zeros and ones
- **过去分词作后置定语：** known as bits
- **重点词汇：** represent（表示）、binary form（二进制形式）、utilize（利用）、combination（组合）、bit（比特/位）

> 💡 binary 的词根 bi- 表示"二"。bit 是 binary digit 的缩写，意思是"二进制位"。它是计算机中最小的信息单位。无论是文字、图像、音频还是程序指令，最底层最终都要转换成 bit 的组合。

## 📄 段落精读

### Computer Hardware

A computer is a machine that can be instructed to carry out sequences of arithmetic or logical operations automatically via computer programming. Modern computers have the ability to follow generalized sets of operations, called programs. These programs enable computers to perform an extremely wide range of tasks.

A "complete" computer including the hardware, the operating system, and peripheral equipment required and used for "full" operation can be referred to as a **computer system**. This term may as well be used for a group of computers that are connected and work together, in particular a **computer network** or **computer cluster**.

> 📌 **核心要点：**
> - **计算机本质：** 一种可以通过编程被指示自动执行算术或逻辑运算序列的机器
> - **计算机系统：** 包含硬件、操作系统和外围设备的"完整"计算机
> - **计算机网络/集群：** 多台计算机连接并协同工作

Computers are used as control systems for a wide variety of industrial and consumer devices. This includes simple special-purpose devices like microwave ovens and remote controls, factory devices such as industrial robots and computer-aided design, and also general-purpose devices like personal computers and mobile devices such as smartphones. The Internet is run on computers and it connects hundreds of millions of other computers and their users.

Conventionally, a modern computer consists of at least one processing element, typically a central processing unit (CPU), and some form of memory. The processing element carries out arithmetic and logical operations, and a sequencing and control unit can change the order of operations in response to stored information. Peripheral devices include input devices (keyboards, mouse, etc.) and output devices (monitor screens, printers, etc.). Peripheral devices allow information to be retrieved from an external source and they enable the result of operations to be saved and retrieved.

---

### Software

Software is a general term for the various kinds of programs used to operate computers and related devices. (The term hardware describes the physical aspects of computers and related devices.) Software can be thought of as the variable part of a computer and hardware the invariable part.

Software is often divided into:
- **Application software:** programs that do work users are directly interested in
- **System software:** includes operating systems and any program that supports application software

The term **middleware** is sometimes used to describe programming that mediates between application and system software or between two different kinds of application software (for example, sending a remote work request from an application in a computer that has one kind of operating system to an application in a computer with a different operating system).

An additional and difficult-to-classify category of software is the **utility**, which is a small useful program with limited capability. Some utilities come with operating systems. Like applications, utilities tend to be separately installable and capable of being used independently from the rest of the operating system.

**Applets** are small applications that sometimes come with the operating system as "accessories". They can also be created independently using Java or other programming languages.

Software can be acquired as:
| 类型 | 英文 | 说明 |
|:---|:---|:---|
| 共享软件 | shareware | 通常试用期结束后需付费 |
| 精简版软件 | liteware | 禁用了部分功能的共享软件 |
| 免费软件 | freeware | 免费但受版权限制 |
| 公有领域软件 | public domain software | 完全免费且无任何限制 |
| 开源软件 | open source | 提供源代码，用户同意不限制改进版本的分发 |
